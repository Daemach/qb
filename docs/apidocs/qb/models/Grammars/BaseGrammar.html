qb.models.Grammars

Class BaseGrammar

lucee.Component
    qb.models.Grammars.BaseGrammar
Direct Known Subclasses:
MSSQLGrammar , MySQLGrammar , OracleGrammar , PostgresGrammar

Grammar represents a platform to run sql on. This is the Base Grammar that other grammars can extend to modify the generated sql for their specific platforms.
